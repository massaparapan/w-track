# WTrack (Weight Track)

Una aplicación web progresiva (PWA) moderna y elegante para llevar un seguimiento diario de tu peso. Desarrollada con Vue.js, sin necesidad de login ni base de datos externa - todos los datos se guardan localmente en tu dispositivo.

## Tecnologías

- **Vue.js 3** - Framework JavaScript progresivo
- **Vite** - Build tool y dev server ultrarrápido
- **Chart.js** - Librería para gráficos interactivos
- **Vue ChartJS** - Integración de Chart.js con Vue
- **Vite PWA Plugin** - Soporte para Progressive Web Apps
- **Workbox** - Service Worker para funcionalidad offline

## Uso

1. **Agregar peso**: Ingresa tu peso en kilogramos en el formulario y presiona "Guardar Peso"
2. **Ver evolución**: El gráfico muestra automáticamente la evolución de tu peso a lo largo del tiempo
3. **Revisar historial**: Desplázate hacia abajo para ver todos tus registros ordenados por fecha
4. **Eliminar registro**: Desliza un registro hacia la izquierda (o arrastra con el mouse) para eliminarlo

Para instalar la app en tu dispositivo móvil:
1. Abre la aplicación en tu navegador
2. Busca la opción "Agregar a pantalla de inicio" o "Instalar app"
3. ¡Listo! La app funcionará como una aplicación nativa
